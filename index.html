视频导出不能直接双击 HTML 使用
图片裁切可以做成普通 HTML，但视频裁切需要 FFmpeg.wasm 多线程环境，浏览器必须从带响应头的本地服务或线上域名打开。

本地测试请在这个目录运行 npm run dev，然后打开 http://localhost:4173。 部署上线时使用项目里的 Netlify / Vercel 配置即可。
